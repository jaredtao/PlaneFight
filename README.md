## 简介：

飞机大战- 王校长吃面包版

蹭个热度，比较粗糙，没有精雕细琢。
## 效果图：

![Demo](Demo/demo.png)

## 功能：

* 点击开始按钮开始游戏
* 键盘左右键控制王校长左右移动
* 空格键发射面包

## 开发环境

* Qt 5.9.x Windows/Ubuntu
